## Installation

git clone https://github.com/RcdsDw/bot_discord.git

npm install

## Lancement

npm start

## Configuration

Créer un fichier .env avec les variables d'environnement suivantes :

TOKEN_BOT=TOKEN_BOT
DATABASE_URL=DATABASE_URL

## Vérification

Pour vérifier le code avant de push, lancer la commande :

npm run checkall
